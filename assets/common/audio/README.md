# Shared Audio Assets

Place reusable audio for all card games here.

`SoundManager` automatically uses these optional SFX files when present:

- `sfx/card_click.ogg`
- `sfx/card_play.ogg`
- `sfx/card_deal.ogg`
- `sfx/bid.ogg`
- `sfx/trick_win.ogg`
- `sfx/level_up.ogg`
- `sfx/game_over.ogg`

If a file is missing, `SoundManager` falls back to generated sounds.
